# 即将推出：mini-SwitchHosts 3.0 Pro 版本

![Logo](../logo/logo.png)

我们正在开发功能更强大的 mini-SwitchHosts 3.0 Pro 版本，预计将在不久的将来与大家见面。

## 3.0 Pro 版本主要新功能预告

### 1. 插件系统

- 支持社区开发的插件扩展
- 内置插件管理器
- 在线插件仓库支持

### 2. 增强的规则管理

- 自定义规则集功能
- 规则分组和标签管理
- 高级规则过滤和优先级设置

### 3. 用户界面升级

- 参考 PyCharm Git 界面设计风格，提供更清晰、专业的用户体验
- 支持暗色主题模式
- 可自定义的界面布局

### 4. 性能优化

- 更快的规则解析和应用速度
- 更低的内存占用
- 改进的并发处理机制

### 5. 平台支持扩展

- 更好的 Windows 11 兼容性
- 扩展的 Linux 发行版支持
- 全面适配 macOS 最新版本和 Apple Silicon 芯片

### 6. 安全性增强

- 规则源数字签名验证
- 增强的备份和恢复机制
- 改进的权限管理

## 预计发布时间

敬请期待，具体发布时间将在确定后公布。
