# Beta 版本说明

此目录包含 mini-SwitchHosts 的测试版本，具有实验性功能。

## V3.0 版本主要升级方向

### 1. 核心功能改进
- **增强的IP解析算法**：采用更先进的DNS查询技术，提高IP地址获取的准确性和速度
- **智能规则过滤**：新增智能算法，自动过滤无效或低质量的Hosts规则
- **增量更新机制**：只更新变化的部分，减少系统资源消耗，提高更新效率

### 2. 用户体验优化
- **现代化UI界面**：重新设计的用户界面，参考PyCharm的Git界面风格，更加清晰专业
- **实时状态监控**：新增状态栏显示实时更新进度和系统状态
- **一键备份恢复**：简化备份和恢复操作流程，提高数据安全性

### 3. 性能提升
- **启动速度优化**：优化程序启动流程，启动时间缩短30%以上
- **内存占用降低**：重构核心模块，显著降低内存使用量
- **并发处理能力**：支持多线程并发处理多个域名解析请求

### 4. 平台兼容性增强
- **Windows系统适配**：针对Windows 11进行了特别优化和适配
- **Linux发行版支持**：扩展支持更多Linux发行版，包括Ubuntu、CentOS、Debian等
- **macOS兼容性**：全面适配macOS最新版本，支持Apple Silicon芯片

### 5. 安全性加强
- **HTTPS规则源**：所有规则源均使用HTTPS加密连接，防止中间人攻击
- **数字签名验证**：关键更新文件增加数字签名验证机制
- **权限管理优化**：精细化权限控制，仅在必要时请求管理员权限

## 注意事项

这些是测试版本，可能包含未预期的行为或错误。请在使用前备份重要数据。
建议仅在测试环境中使用，生产环境请使用稳定版本。
