## Logo 含义

### 设计理念
mini-SwitchHosts 的 Logo 设计融合了以下核心概念：

1. **开关元素** - 代表"Switch"概念，表示切换和改变网络配置的能力
2. **网络连接** - 象征网络优化和连接管理功能
3. **盾牌保护** - 体现安全性和对系统配置的保护作用
4. **现代化设计** - 反映项目的现代技术基础和用户体验

### 设计元素
- **主图形**: 抽象化的开关与网络节点结合
- **颜色**: 科技蓝为主色调，表达技术感和可靠性
- **字体**: 现代简洁的无衬线字体，易于识别

## 使用规范

### 颜色使用
1. **主色调**: 科技蓝 (#2196F3)
2. **辅助色**: 白色 (#FFFFFF) 或深灰 (#333333)
3. **背景适应**: 根据背景色选择合适的 Logo 颜色版本

### 尺寸要求
1. **最小显示尺寸**: 不小于 32x32 像素
2. **推荐显示尺寸**: 128x128 像素或更大
3. **矢量使用**: 优先使用 SVG 格式以保证清晰度

### 使用场景
1. **软件界面**: 程序标题栏、关于对话框
2. **文档**: README 文件、用户手册
3. **网站**: 项目主页、GitHub 页面
4. **宣传材料**: 演示文稿、技术博客

### 禁止行为
1. 修改 Logo 的颜色、比例或结构
2. 在复杂背景上使用导致难以识别
3. 未经许可用于商业产品或其他项目
4. 扭曲、旋转或变形 Logo
